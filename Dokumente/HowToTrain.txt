
1. Start Venv: 
venv\Scripts\activate



2. Run Automatic training :


python train_automation.py

python new_automation.py


3. View results with TensorBoard:
tensorboard --logdir results



4. start manual training:

mlagents-learn config\configDatei.yaml  --run-id=(Name) --env="Builds\**.exe" 



Extensions for 4 manual training command :

1.  --initialize-from=(name_in_Results_Folder)   // von einem vorherigen Training starten/ableiten


2. --inference  // nur ausführen, kein Training

3. --force  // überschreibt vorhandene RunID Ordner

4. --resume  // setzt ein vorheriges Training fort





//fertiges Modell laufen lassen ohne Training,in Trainingsumgebung für vergleich zu generalisierung :
